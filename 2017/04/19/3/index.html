GIT新手教程及国内镜像安利

2017-04-19
git

博主在搭建博客过程中，对GIT的下载安装深感苦恼，在不翻墙情况下想下载GIT，真的很慢很慢。。。
好在有大佬贡献出了国内镜像，在此安利一波：
http://gitb.org/gcc-for-windows/
另外附上一波GIT的入门教程（附安装方法）
http://www.bootcss.com/p/git-guide/

但还有一个不幸的消息，github desktop离线安装版并未找到合适的资源，最好还是翻墙下载。虽然这玩意很鸡肋，但是我不得不用，在手贱更新了win10的创意者更新后，通过git部署到github的方式挂掉了，至今未找到原因，很无奈。。。急切等待备案的完成，到那时期中考试结束，就有时间将其部署到阿里云的服务器上，博客也将迎来大改版啦。
